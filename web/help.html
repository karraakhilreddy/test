Q2: WHEN DOES STATUS IN COURSE DETAILS PAGE CHANGE?

ANS :

  THE STATUS CHANGES IN FIVE STAGES
  1) WHEN YOU SELECT THE COURSE, THE STATUS CHANGES TO INSPECTION FEE PENDING
  2) ONCE YOU PAY THE INSPECTION FEE, THE STATUS CHANGES TO INSPECTION FEE PAID
  3) AFTER THE ADMINISTRATION GRANTS THE PERMISSION , STATUS CHANGES TO INSPECTION SUCCESS
  OR ELSE IF THEY DENY THE PERMISSION THE STATUS CHANGES TO INSPECTION FAILED
  NOTE : YOU NEED TO PAY AFFILIATION FEE FOR THE COURSES WITH INSPECTION SUCCESS STATUS
  4) AFTER THE PAYMENT OF THE AFFILIATION FEES THE STATUS CHANGES TO AFFILIATION PENDING
  5) AFTER THE ADMINISTRATION GRANTS THE PERMISSION , STATUS CHANGES TO AFFILIATION SUCCESS
OR ELSE IF THEY DENY THE PERMISSION THE STATUS CHANGES TO AFFILIATION FAILED
